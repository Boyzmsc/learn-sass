## Compile

### Parcel
```bash
$ npm init -y
$ npm i -g parcel-bundler
$ npm i --save-dev node-sass
```

```html
<link rel="stylesheet" href="scss/main.scss">
```

```bash
$ parcel index.html
# or
$ parcel build index.html
```
